Tags HTML são palavras-chave usadas em HTML para exibir páginas da web com certas propriedades. Eles também são usados para definir elementos HTML. Um elemento HTML consiste em uma tag inicial, algum conteúdo e uma tag final. O navegador da Web lê o documento HTML de cima para baixo,da esquerda para a direita. Cada tag HTML define uma nova propriedade que ajuda na renderização do site.

Marcas HTML

Os colchetes '<>' contêm uma tag HTML. Existem dois tipos de tags HTML - tags vazias ou tags singleton e tags de container. As tags singleton ou tags vazias não contêm nenhum conteúdo, como uma imagem ou um parágrafo e, portanto, não precisam ser fechadas, enquanto as tags container devem ser fechadas...
